## 短路表达式

> 短路表达式： 作为`&& 和 ||`操作符的操作数表达式，这些表达式在进行求值时，只要最终的结果已经可以确认是真或假，求值过程便告终止，这称之为短路求值。这是这两个操作符的一个重要属性。

```javascript
  // || 短路表达式
  var foo = a|| b;
  // 相当于
  if(a){
    foo = a;
  }else{
    foo = b;
  }
  
  // && 短路表达式
  var bar = a && b;
  // 相当于
  if(a){
    bar = b;
  }else{
    bar = a;
  }
```
